Пример Абстрагирования.

В разработке некоторой графической библиотеки имеется базовый класс "BaseShape" для графических элементов.
Он имеет методы для рисования, изменения размера, перемещения и т.д.

Класс "Triangle" — класс для рисования прямоугольников, наследуется от базового класса и имеет дополнительный метод(ы) необхоимые для рисования прямоугольников.
Класс "Circle" — класс для рисования кругов, также наследуется от класса "BaseShape" и имеет метод(ы) для рисования кругов.

```
     BaseShape (Base Class)
     ^           ^
     |           |
     Rectangle   Circle
```

Пример Факторизации — Разработка системы автоматизации управления складом.

Класс "Product" — базовый класс для продуктов на складе, содержит информацию о наименовании, количестве, цене и т.д.

Класс "BoxedProduct" — класс для продуктов, которые могут храниться в коробках, наследуется от класса "Product" и добавляет информацию о размере коробки.
Класс "BarellProduct" — класс для продуктов, которые могут храниться в бочках, наследуется от класса "Product" и добавляет информацию, например, о объеме бочки.

Здесь факторизация происходит за счет того, что классы "BoxProduct" и "BarellProduct" являются различными специализациями базового класса Product и добавляют специфичные для каждого типа продукта свойства, на самом деле являясь частными случаями "Product"

```
     A (Product)
     ^           ^
     |           |
     B   (Box Product)  C (Barrel Product)
```
